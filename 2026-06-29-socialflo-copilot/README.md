# SocialFlo — Your social AI co-pilot (Demo)

Konzept-Demo für **Vidushi Malhan** (Brand & Communication @ SocialFlo) — gebaut, um in wenigen Minuten zu zeigen, wie schnell sich mit nyxory eine echte, live deploybare App bauen lässt.

## Ziel

Ein gebrandetes „Social AI Co-Pilot"-Dashboard im SocialFlo-Look (Lila/Teal, Wave-Logo, Smiley-Maskottchen), das den Kern der Idee greifbar macht: **Content at Scale, ohne Team-Burnout.**

## Was die Demo zeigt

- **Content Co-Pilot** (das Highlight): interaktiver KI-Composer. Thema + Tonalität wählen → der Co-Pilot „tippt" in Echtzeit eine fertige Instagram-Caption inkl. Hashtags. (Simulierte KI, vorlagenbasiert — keine API nötig.)
- **KPI-Leiste**: Reichweite, Engagement-Rate, geplante Posts, gesparte Stunden.
- **Content-Kalender**: Wochen-Queue über Instagram, LinkedIn, TikTok, X mit Status (Geplant / Entwurf / Live).
- **Engagement-Sparkline** + **Top-Post** der Woche.

## Setup

Eine einzelne, eigenständige Datei — keine Build-Tools, keine Dependencies.

```
open index.html
```

## Hinweise

- Alle Zahlen, Posts und Texte sind **fiktiv**.
- Single-File `index.html` (HTML/CSS/JS), responsive, dark.
- Live deploybar über nyxory als statische Seite.
