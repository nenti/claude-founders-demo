# 2026-07-02 · Mobile Cloud Code Workshop

Testlauf für ein Workshopformat zum Thema **Claude Code vom Handy aus in der Cloud**
(geplant mit Vitoshi).

## Ziel

Kurz ausprobieren, wie eine Mobile-Cloud-Code-Session abläuft: vom Handy eine
Session starten, Claude im Cloud-Container eine einfache Seite bauen lassen und
das Ergebnis committen & pushen.

## Setup

- Reine statische Seite, keine Abhängigkeiten.
- Lokal öffnen: `index.html` im Browser öffnen, oder
  `python3 -m http.server` im Ordner starten und `http://localhost:8000` aufrufen.

## Ergebnis

`index.html` — eine kleine animierte "Hallo Welt"-Seite, die gleichzeitig die
vier Schritte des Workshop-Ablaufs zeigt.
