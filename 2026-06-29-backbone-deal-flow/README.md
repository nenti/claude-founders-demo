# Deal Flow Cockpit — Demo für BackBone Ventures

Schlanke Konzept-Demo: ein **Deal-Flow / Portfolio-Cockpit**, wie es für einen
VC-Partner (z. B. Philippe Bernet, Partner @ BackBone Ventures) im Alltag
relevant ist.

## Ziel

Auf einen Blick zeigen, was einen Partner interessiert:

- **KPIs** — neue Deals, Deals in Due Diligence, Dry Powder, Time-to-IC.
- **Top Opportunities** — die heissesten Startups nach Fit-Score, mit Sektor,
  Runde und aktueller Stage.
- **Pipeline-Funnel** — von Inbound/Screening bis Term Sheet inkl. Konversion.

## Setup

Eine einzige, eigenständige Datei — kein Build, keine Dependencies.

```
open index.html
```

(oder doppelklicken)

## Hinweis

Alle Zahlen und Startups sind **fiktive Demo-Daten** im BackBone-Branding
(Grün/Schwarz). Keine echten Deals.
